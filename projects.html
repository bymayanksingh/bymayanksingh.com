Metroworks CLI→

💡 It is a command line based utility which helps in finding and effectively visualizing shortest route between intra delhi metro stations, It uses dijkstras algorithm.


👨‍💻 As a developer, I implemented the dijkstras algorithm using standard template library in C++, along with that I along with my team worked with graphviz which is a graph visualization text based language, we integrated it with our C++ code to dynamically visualize path between metro stations.


⚙️ Used C++, graphviz, fim, zenity, sl, notify-osd


📅 Built in Oct 2018
